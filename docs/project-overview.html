Core Features
F-01 to F-12 — Full Feature Map
F-01 Multi-Tenant Auth (SSO) M2
SAML 2.0 / OIDC with Azure AD & Google Workspace. MFA enforcement per tenant. RBAC with 33 granular permissions.
✓ Login < 2s · MFA enforced · Tenant isolation verified
F-02 Financial Ledger (GL) M3
Double-entry accounting engine. Multi-currency with daily FX auto-fetch from ECB. Period close with role-based override.
✓ Zero unbalanced entries · FX auto-fetched · Period lock enforced
F-03 AP / AR Automation M3
Invoice OCR with 95%+ accuracy. 3-way PO/GR/Invoice matching. Automated payment runs & aging reports.
✓ OCR ≥ 95% · Auto-match in < 30s · Aging reports live
F-04 HR & Payroll Engine M4
Employee lifecycle management. Leave & attendance tracking. Gross-to-net payroll with configurable tax slabs. Payslip PDF generation.
✓ Payroll < 5min for 10k employees · Full audit trail
F-05 Supply Chain & Inventory M5
PO lifecycle from requisition to receipt. Vendor portal API. Real-time stock levels with FIFO costing. Automated reorder triggers.
✓ Reorder at configurable threshold · Vendor email/webhook alert
F-06 AI Demand Forecasting M5
Python FastAPI ML microservice. Prophet model for time-series SKU forecasting. LSTM (PyTorch) for high-volume SKUs. Weekly retraining via BullMQ.
✓ MAPE < 12% on 90-day horizon · Retrain weekly
F-07 Project Management M4
Gantt chart with drag-to-reschedule. Resource allocation & utilisation heatmap. Budget tracking with variance alerts at 10% overrun.
✓ Overrun alert at 10% · Gantt renders < 1s
F-08 Business Intelligence M6
Drag-and-drop dashboard builder. Recharts + ECharts widgets (bar, line, pie, heatmap). Drill-down analytics. Scheduled reports via email.
✓ Dashboard saved < 500ms · Exports PDF/Excel
F-09 Audit & Compliance Log M2
Immutable audit trail with SHA-256 hash chaining (tamper detection). GDPR data subject request API. All mutations logged with before/after states.
✓ Tamper-evident · DSR fulfilled < 72h
F-10 Notification Engine M5
Multi-channel: in-app (SSE), email (AWS SES), SMS (Twilio), webhook (HMAC). Per-user channel preferences. BullMQ retry with dead-letter queue.
✓ Delivery confirmed · 3x retry on failure · Channel preference
F-11 API Gateway & Webhooks M1+M2
REST (OpenAPI 3.1) + GraphQL (Apollo v4) gateway. Outbound webhook subscriptions with HMAC signing. All endpoints versioned at /api/v1.
✓ OpenAPI 3.1 spec published · All endpoints versioned
F-12 Offline / PWA Support M6
Service worker caching for critical read views (dashboard, inventory, leave). Background sync queue on reconnect. Installable PWA manifest.
✓ Core views offline · Sync without data loss on reconnect

Quick Start
Get Running in 5 Commands
# 1. Clone the repo
git clone <repo-url> && cd amdox-erp

# 2. Copy environment variables
cp .env.example .env

# 3. Start PostgreSQL, Redis, Keycloak, Elasticsearch
docker-compose up -d

# 4. Install, migrate & seed
cd backend && npm install && npx prisma migrate deploy && npm run seed

# 5. Start dev servers
npm run dev   # runs frontend :3000 + backend :5000 concurrently
