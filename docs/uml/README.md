### 1. Install the PlantUML Extension in VS Code

* Open **VS Code**
* Press **`Ctrl + Shift + X`** to open the Extensions panel.
* Search for **PlantUML**.
* Install the PlantUML extension.

### 2. Preview the Diagram

* Create or open a `.puml` or `.plantuml` file.
* Write your PlantUML code.
* Click the **Preview** icon in the **top-right corner** of VS Code (or use the PlantUML preview command) to render the diagram.

### 3. Use the Online PlantUML Server (Alternative)

If you don't want to install the extension, you can paste your PlantUML code into the official online server:

**[PlantUML Online Server](https://plantuml.com/en/server?utm_source=chatgpt.com)**

From there, you can:

* Paste your PlantUML code.
* Generate and preview the diagram instantly.
* Export or share the generated diagram.
* Use it for documentation in your GitHub README or project wiki. ([PlantUML.com][1])
